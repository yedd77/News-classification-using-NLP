News Catergorization is a web-based application using Flask that combines a chatbot and a text categorization tool. The chatbot interacts with users by analyzing their input and providing appropriate responses based on predefined intents and answers stored in a file. For example, if a user greets the chatbot, it recognizes the intent and responds with a suitable greeting. If the chatbot cannot understand the input, it suggests rephrasing the query. On the other hand, the text categorization tool processes and classifies text, such as news articles, into categories like sports or politics. Before categorization, it cleans the input by removing unnecessary elements like links, hashtags, or special characters and ensures the text is in English.

The application utilizes machine learning models to perform its tasks. One model is specifically trained to identify the intent of the userâ€™s input, while another, a pre-trained BERT model, classifies the news text into categories. The application handles errors gracefully, such as informing the user when the input language is unsupported or when the text is invalid. It includes a web interface where users can interact with the chatbot by typing messages or use the categorization feature by pasting text to analyze. The application runs on a local web server, making it accessible via a web browser for testing and use. This tool is an excellent example of combining artificial intelligence, natural language processing, and web development to create an intuitive and functional user experience.
